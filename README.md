# Wariable Digital | Agencia de Desarrollo y Automatización - Landing Page v1.0

¡Bienvenido al repositorio oficial de la Landing Page de **Wariable Digital**! Esta es una plataforma web moderna, optimizada y de alta conversión diseñada para promocionar nuestros servicios de transformación digital, desarrollo de software a medida y automatización de procesos comerciales.

Nuestra propuesta de valor se centra en resolver la pérdida de tiempo en tareas repetitivas mediante soluciones tecnológicas estratégicas e infraestructura digital escalable.

---

## 🚀 Características Principales

*   **Diseño Premium (Dark Mode):** Interfaz moderna y minimalista con paleta de colores oscuros, violetas y azules eléctricos que transmiten innovación y tecnología.
*   **Sección de Puntos de Dolor:** Identificación clara de las problemáticas de los negocios (pérdida de turnos, atención lenta en WhatsApp, falta de presencia en Google).
*   **Catálogo de Servicios:** Bloques interactivos que detallan la oferta principal de nuestra agencia.
*   **Llamados a la Action (CTA) Claros:** Botones estratégicos orientados a la conversión directa ("Quiero una asesoría gratis" e "Iniciar consultoría estratégica").
*   **Diseño Responsivo:** Totalmente adaptado para visualizarse de forma óptima en dispositivos móviles, tablets y computadoras de escritorio.

---

## 🛠️ Servicios Promocionados por Wariable Digital

1.  **Webs & Landing Pages:** Desarrollo de vidrieras digitales estéticas, rápidas y optimizadas para la venta con un solo clic.
2.  **Agentes de IA para WhatsApp & Automatización:** Sistemas inteligentes que agendan turnos, responden dudas frecuentes y organizan pedidos en piloto automático sin intervención humana.
3.  **Sistemas a Medida & Gestión de Clientes:** Conexión de herramientas comerciales (Bases de datos, Excel, APIs de comunicación y CRMs) para que los negocios funcionen en piloto automático.

## ⚙️ Instalación y Uso Local

Si quieres clonar este proyecto y ejecutarlo en tu entorno local, sigue estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/jeremias1234-bit/wariable-lanfing.git
   ```

2. **Navegar al directorio del proyecto:**
   ```bash
   cd TU_REPOSITORIO
   ```

3. **Abrir el proyecto:**
   * Si utilizas VS Code, ejecuta:
     ```bash
     code .
     ```
   * O simplemente haz doble clic sobre el archivo `index.html` en tu gestor de archivos para desplegarlo en el navegador.
